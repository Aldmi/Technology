*** созадть шаблон проекта - ставит package.json
npm init -y 

*** Поставить зависимость - библиотеку typescript
    Возможно придется уставноить TS глобавльно
	Убедится что TS доступен глобально 
    tsc -v	
	Если версия локальная (смотерть в package.json) отличается от глобальной то обновить глобальную версию
	npm i -g typescript	
npm install -D typescript

*** Компилятор TS
1. Установить tsconfig.json - файл настройки компилятора
   tsc --init
   
2. Запуск компилятора в ручную
   tsc
  
3. Запуск компилятора при изменении файла
   tsc --watch
